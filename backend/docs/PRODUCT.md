# Product

JobRadar AI is a job-search workspace for software developers.

It centralizes job opportunities, application tracking, search automation, and future AI assistance into one product designed for real job-search execution.

## Current User

The initial user is a software developer actively looking for a job.

This user may be:

- Applying to many roles at once.
- Tracking opportunities manually in spreadsheets.
- Searching across multiple platforms.
- Using AI tools to understand job descriptions or tailor applications.
- Trying to keep follow-ups, notes, and application status organized.

## Problem

The job search is scattered across too many tools.

A developer may use:

- LinkedIn for discovery.
- Greenhouse and Lever for applications.
- Gmail for recruiter communication.
- Excel or Google Sheets for tracking.
- Notes apps for context.
- ChatGPT for resume tailoring and preparation.
- Browser tabs for company research.

This creates duplicated work, missed opportunities, poor visibility, and unnecessary stress.

## Solution

JobRadar AI provides one place to manage the process.

The current backend already supports saving jobs, listing them, updating status and metadata, deleting jobs, searching jobs, and saving search results.

The product should grow from that foundation into a complete workflow system.

## Main Features

Current foundation:

- Save job opportunities.
- List saved jobs.
- View one job.
- Update job metadata and status.
- Delete jobs.
- Search for jobs.
- Search and save new jobs while avoiding duplicate URLs.

Planned capabilities:

- Job pipeline board.
- Better filtering and search.
- AI job-fit scoring.
- Resume-to-job matching.
- Application history.
- Follow-up reminders.
- Gmail integration.
- Company research summaries.
- Exportable reports.
- Job-search analytics.
- Frontend dashboard.

## Future Vision

JobRadar AI should become the command center for a developer's job search.

The future product should help answer:

- What jobs should I apply to today?
- Which applications need follow-up?
- Which roles match my profile best?
- Which companies are worth prioritizing?
- What should I improve in my resume?
- Where am I spending time without results?
- What changed since my last search session?

The product should make job searching calmer, more organized, and more effective.
